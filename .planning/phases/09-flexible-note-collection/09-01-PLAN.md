**CollectionSpec type (discriminated union per research recommendation):**
```typescript
import { App, TFile, getAllTags } from 'obsidian';

export type CollectionSpec =
  | { mode: 'folder'; folderPath: string }
  | { mode: 'tag'; tag: string }
  | { mode: 'links'; rootFile: TFile; depth: 1 | 2 }
  | { mode: 'note'; file: TFile };
```

**isSelfTestFile(file: TFile): boolean**
- Return true if `file.basename === '_self-test'`
- Return true if `file.path.startsWith('_self-tests/')`
- Return true if `file.basename.endsWith('_self-test')`
- Return false otherwise

**collectNotesByTag(app: App, tag: string): TFile[]**
- Normalize target: `tag.replace(/^#/, '').toLowerCase()`
- Iterate `app.vault.getFiles()`, filter to `.md` extension
- Skip files where `isSelfTestFile(file)` returns true
- For each file: `app.metadataCache.getFileCache(file)`, then `getAllTags(cache) ?? []`
- Normalize each tag with `t.replace(/^#/, '').toLowerCase()`
- Match if `normalized === normalizedTarget || normalized.startsWith(normalizedTarget + '/')`  (per D-03 hierarchical matching)
- Return matched TFile[]

**getAllVaultTags(app: App): Map<string, number>**
- Iterate all `.md` vault files
- For each: getFileCache, getAllTags, normalize, increment count in Map
- Return Map

**collectNotesByLinks(app: App, rootFile: TFile, depth: 1 | 2): TFile[]**
- BFS with visited Set initialized with rootFile.path
- Queue items: `{ path: string, d: number }`
- Start with `[{ path: rootFile.path, d: 0 }]`, result `[rootFile]`
- While queue not empty: shift item, if `item.d >= depth` continue
- Get `app.metadataCache.resolvedLinks[item.path] ?? {}`
- For each key (destination path): skip if visited, get file via `app.vault.getFileByPath(linkedPath)`
- Skip if null, skip if extension !== 'md', skip if isSelfTestFile
- Add to visited, result, and queue with `d: item.d + 1`

**buildTagOutputPath(tag: string): string**
- `const normalized = tag.replace(/^#/, '');`
- Return `_self-tests/tags/${normalized}.md`

**buildLinksOutputPath(rootFile: TFile): string**
- Return `_self-tests/links/${rootFile.basename}.md`

**buildNoteOutputPath(file: TFile, mode: 'same-folder' | 'centralized'): string**
- If `mode === 'same-folder'`:
  - `const folder = file.parent?.path ?? '';`
  - Return `folder ? folder + '/' + file.basename + '_self-test.md' : file.basename + '_self-test.md'`
- If `mode === 'centralized'`:
  - `const safeName = file.path.replace(/\.md$/, '').replace(/\//g, '-');`
  - Return `_self-tests/notes/${safeName}.md`

**writeOutputToPath(app: App, filePath: string, content: string): Promise<void>**
- Split path by `/`, iterate segments to build cumulative directory paths
- For each dir segment (excluding the filename): check `app.vault.getAbstractFileByPath(dir)`
- If null, call `await app.vault.createFolder(dir)`
- After dirs created: check `app.vault.getFileByPath(filePath)`
- If exists: `await app.vault.modify(existing, content)`
- If not: `await app.vault.create(filePath, content)`

**buildFrontmatter(spec: CollectionSpec, collectedFiles: TFile[]): string**
- Build `source_notes` array: `collectedFiles.map(f => '"[[' + f.basename + ']]"').join(', ')`
- Determine `mode` and `source` from spec:
  - `tag` -> source is spec.tag
  - `links` -> source is spec.rootFile.basename
  - `note` -> source is spec.file.basename
  - `folder` -> source is spec.folderPath
- Return joined lines:
  ```
  ---
  last_review: null
  next_review: null
  review_count: 0
  review_interval_days: 1
  source_mode: <mode>
  source: "<source>"
  source_notes: [<sourceNotes>]
  ---
  ```
  Followed by a blank line.
