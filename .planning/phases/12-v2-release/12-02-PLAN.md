**Section 3: How to use (per D-09)**
- Heading: `## How to use`
- Brief intro sentence: "Self Test supports four ways to generate questions from your notes."
- Four subsections, each with a heading, a brief description of what the mode does, and the entry points:

  `### Folder mode`
  - Generates questions from all notes in a selected folder
  - Entry points: command palette ("Generate Self-Test for Current Folder"), right-click any folder in file explorer ("Generate Self-Test"), sidebar Folders tab
  - Output: `_self-test.md` in the folder

  `### Tag mode`
  - Generates questions from all notes in the vault that share a specific tag
  - Entry points: command palette ("Generate Self-Test by Tag"), sidebar Tags tab
  - Output: `_self-tests/` folder in the vault root

  `### Linked Notes mode`
  - Generates questions from a root note plus all notes it links to, with an optional depth-2 toggle for links of links
  - Entry points: command palette ("Generate Self-Test from Linked Notes"), sidebar Links tab
  - Output: `_self-tests/` folder in the vault root

  `### Single Note mode`
  - Generates questions for one note
  - Entry points: command palette ("Generate Self-Test for Current Note"), right-click any note in file explorer ("Generate Self-Test")
  - Output: `{note-name}_self-test.md` in the same folder as the source note
